# compare display property(inline-block, inline)

## display: inline
* element width, hieght 설정 불가

## idsplay: inline-block
* element width, hieght 설정을 할 수 있다
* top, bottom margins/padding를 설정할 수 있다. 
* element line-break 뒤에 붙이지 않는다.
